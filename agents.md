# El-Fix Website

Mål:
Bygga en modern, snabb och SEO-optimerad hemsida för El-Fix i Sundsvall.

Teknik:
- Statisk webbplats
- HTML
- CSS
- JavaScript
- Ingen WordPress

Design:
- Modern
- Hög trovärdighet
- Elinstallatör
- Företagskunder och privatpersoner

Viktigt:
- Ändra aldrig kontaktuppgifter utan instruktion.
- Ändra aldrig företagsnamn.
- Alla sidor ska fungera på mobil.
- SEO är viktigt.

Sidor:
- Startsida
- Om oss
- Tjänster
- Laddboxar
- Solceller
- Kontakt

När du gör ändringar:
- Beskriv vilka filer som ändrats.
- Motivera större designändringar.
